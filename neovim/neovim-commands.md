# NeoVim Commands

- Legend:

  - `<leader>`: space

- `gr` - Look for references
- `gd` - Look for definition
- `<leader>gf` - Look for git files
- `<leader>sf` - Look for files

- `:Lex` - Open file explorer as a left split
  - `%` - Create file
  - `d` - Create directory
  - `D` - Delete file or directory
- `:NvimTreeToggle` - Toggle file explorer
